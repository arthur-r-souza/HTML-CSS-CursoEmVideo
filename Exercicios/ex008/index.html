Principais formatações

Negrito/destaque

Nesta frase temos um termo em negrito usando a tag <b> (não semântica)

Aqui já temos um termo destacado de maneira semântica com a tag <strong>

Termo em itálico/ênfase

Aqui temos um termo em itálico com a tag <i> (não semântica)

Já aqui temos um termo em ênfase, com a tag <em> (semântica)

Texto marcado

Podemos criar também um texto marcado através da tag <mark>.

Texto grande e pequeno

As tags <big>; e <small>; eram utilizadas antigamente, mas por não possuírem valor semântico, foram descontinuadas. O melhor é alterar através do CSS

Texto deletado

Podemos incluir um texto excluído para indicar que ele não deve mais ser considerado através de <del>

Texto incluído

Podemos marcar um texto como incluído para identificar que ele foi inserido de maneira posterior através de <ins>

Texto sobrescrito

Para inserir expoentes, por exemplo: x20+3 através de <sup>

Texto subescrito

Para inserir índices de elementos químicos, por exemplo H2O através de <sub>
