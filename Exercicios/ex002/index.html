Estudo de parágrafos

Para escrever um parágrafo, basta usar a tag <p>.

No caso de querer quebrar uma linha
basta usar a tag <br>.

Podemos também adicionar alguns símbolos no site, como
® - Marca registrada
© - Copyright
™ - Trademark
€ - euro
£ - Libra
¥ - yen
$ - Dollar sign
¢ - cent
∅ - sinnal de vazio
∑ - Somatório
δ - delta
↑ - seta para cima
← - seta para a esquerda
↓ - seta para baixo
→ - seta para a direita

Para a adição de emojis no corpo do site, basta usar o sinal &#x seguido do código U+ do emoji.
Exemplo: ❤
